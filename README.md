# 🧱 打磚塊遊戲

一款基於網頁的打磚塊遊戲，支援電腦鍵盤與手機觸控操作。

🎮 **線上遊玩：** https://hankchou2004.github.io/breakout-game/

---

## 功能介紹

### 主選單
- 進入遊戲1、遊戲2
- 前往商店購買道具
- 前往主題選擇更換顏色

### 遊戲1
- 可調整球的大小與速度
- 打磚塊得分、自動累積金幣
- 支援鍵盤（← →）與手機觸控左右方向鍵

### 遊戲2
- 磚塊有耐久度，需多次擊中才能破壞
- 難度隨關卡遞增
- 支援鍵盤與手機觸控操作

### 道具系統
| 道具 | 效果 |
|------|------|
| 道具1 | 爆炸彈：擊中磚塊後引發範圍爆炸 |
| 道具2 | 穿透球：發射可穿透磚塊的雷射 |
| 道具3 | 多球：同時發射 20 顆球 |

### 商店
- 使用遊戲中累積的金幣購買道具
- 可購買隨機道具（較划算）

### 主題選擇
- 可自訂背景、球、磚塊、板子、文字的顏色
- 設定儲存於 sessionStorage，切換頁面後仍保留

---

## 操作方式

| 平台 | 操作 |
|------|------|
| 電腦 | `←` `→` 移動板子，`Space` 開始遊戲 |
| 手機 | 畫面上滑動手指移動板子，或使用畫面底部方向鍵 |

---

## 技術說明

- 純前端實作，無需後端伺服器
- 使用 HTML5 `<canvas>` 繪製遊戲畫面
- 使用 ES Module（`import` / `export`）管理各頁面邏輯
- 遊戲資料（金幣、道具、顏色）儲存於 `sessionStorage`
- 響應式設計，畫布尺寸自動適應螢幕大小
- 使用 `@media (pointer: coarse)` 偵測觸控裝置，自動顯示方向鍵

### 檔案結構

```
├── index.html        # 主選單

```
